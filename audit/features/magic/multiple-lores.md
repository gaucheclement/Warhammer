# Magie - Gestion lores multiples

## Vue d'ensemble

Apprentissage lores multiples strictement reglemente. Arcanes interdit (corruption), Divine possible (multi-dieux), Chaos dangereux.

**References** : [domains.md](domains.md), [learning.md](learning.md)

## Magie Arcanes (Huit Vents)

**Regle absolue** : UN domaine UNIQUEMENT, jamais changement

**Interdiction melange** : Melanger Vents differents = corruption immediate, risque devenir Sorcier corrompu
**Specialisation fixe** : Domaine choisi acquisition talent reste definitif
**Progression verticale** : Rangs talent (1-4) augmentent maitrise UN domaine, pas acces autres

**Exemple violation** : Sorcier Feu veut apprendre sorts Bete → IMPOSSIBLE, melange Aqshy + Ghur = Sorcellerie (corruption)

**Lore WFRP** : Colleges Imperiaux interdisent formellement multi-domaines. Magisters pris melanger Vents = execution heresie.

## Magie Divine

**Multi-dieux possible** : Personnage peut servir plusieurs divinites (rare)

**Prerequis** : Talent "Invoke (Dieu)" distinct PAR dieu
**Cout** : Chaque talent cout normal XP
**Acces sorts** : Miracles TOUS dieux servis + benedictions universelles

**Exemples** :
- Pretre sert Sigmar + Ulric : Talents "Invoke (Sigmar)" + "Invoke (Ulric)" → Miracles des deux
- Initie devient Pretre autre dieu : Peut garder ancien + acquerir nouveau

**Restrictions sociales** : Multi-devotion rare, certains cultes jaloux (Sigmarites intolerants autres dieux)

## Magie Chaos

**Changement dieu** : Possible mais perd sorts ancien dieu

**Mecanisme** : Abandon ancien culte, initiation nouveau
**Consequence** : Sorts ancien dieu inaccessibles (faveur perdue)
**Nouveau dieu** : Acces sorts + Indivisibles

**Danger** : Chaque changement accroit Corruption, instabilite

## Magie Naturelle et Sorcellerie

**Magie Naturelle** : Un seul systeme (folklore local)
- Pas lores multiples, tradition unique village/region

**Sorcellerie** : Melange Vents (raison corruption)
- "Lores multiples" = melange dangereux automatique
- +1 Corruption par sort Imparfait
- Exactement ce que Colleges interdisent

## Exemples concrets

**Magister Feu Rang 4** :
- Maitrise complete domaine Feu
- 15-20 sorts Feu connus
- NE PEUT PAS apprendre UN SEUL sort Cieux/Bete/Vie/etc.
- Tentation = corruption, devenir Sorcier proscrit

**Pretre Sigmar + Ulric** :
- Rare mais possible (devotion exceptionnelle)
- Talent "Invoke (Sigmar)" : Miracles Sigmar
- Talent "Invoke (Ulric)" : Miracles Ulric
- Toutes benedictions accessibles (universelles)
- Cout total : 2 talents + XP miracles chaque dieu

**Cultiste Nurgle devientTzeentch** :
- Abandonne Nurgle, sorts Nurgle perdus
- Initie Tzeentch, sorts Tzeentch accessibles
- Corruption accrue changement
- Sorts Indivisibles conserves (pas lies dieu specifique)

**Sorciere melange Vents** :
- Talent "Witchcraft" = melange automatique
- Pas "lores multiples" proprement dit
- Corruption systematique
- Chassee Repurgateurs

## Couts additionnels

**Arcanes** : N/A (impossible)
**Divine multi-dieux** : Cout talent additionnel "Invoke (Dieu 2)" + XP miracles chaque dieu
**Chaos changement** : Cout nouveau talent + perte sorts anciens

## Restrictions WFRP

**Articles Articles Sorcellerie (Empire)** :
- Article Premier : Seuls Colleges Imperiaux pratiquent magie legalement
- Article Deuxieme : Melange Vents = sorcellerie proscrite
- Punishment : Bucher

**Colleges Imperiaux** :
- Formation mono-domaine stricte
- Surveillance mutuelle entre Ordres
- Magisters multi-domaines = traitres, execution

## Regles coeur

1. Arcanes UN domaine UNIQUEMENT : Melange = corruption, impossible
2. Divine multi-dieux possible : Talent distinct requis par dieu
3. Rangs talent vertical : Maitrise domaine, pas nouveaux domaines
4. Chaos changement dieu : Perd sorts ancien
5. Sorcellerie = melange interdit : Corruption systematique
6. Benedictions universelles : Accessibles tous pretres independamment nombre dieux
