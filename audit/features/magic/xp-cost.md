# Magie - Cout XP sorts

## Vue d'ensemble

Cout XP apprentissage sorts varie selon CN (Casting Number), type magie, appartenance carriere et lore.

**References** : [learning.md](learning.md), [casting-number.md](casting-number.md)

## Formules cout base

### Arcanes (selon CN)
**Dans carriere** : CN × 10 XP
**Hors carriere** : CN × 20 XP (double)

**Exemples** :
- Sort CN 2 : 20 XP (carriere) / 40 XP (hors)
- Sort CN 5 : 50 XP (carriere) / 100 XP (hors)
- Sort CN 8 : 80 XP (carriere) / 160 XP (hors)

### Divine (fixe)
**Benedictions** :
- Dans carriere : 50 XP
- Hors carriere : 100 XP

**Miracles** :
- Dans carriere : 100 XP
- Hors carriere : 200 XP

### Autres types
**Magie Mineure** : 30 XP (carriere) / 60 XP (hors)
**Magie Naturelle** : 30 XP + composants (carriere) / 60 XP + composants (hors)
**Chaos** : CN × 10 (toujours hors carriere Empire, illegal)

## Impact carriere

**Dans carriere magique** : Cout normal
- Apprenti Sorcier, Sorcier : Sorts domaine CN × 10
- Initie, Pretre : Benedictions 50, Miracles 100
- Herboriste : Naturelle 30 XP

**Hors carriere** : Cout DOUBLE
- Erudit apprend sorts : CN × 20
- Noble apprend benedictions : 100 XP

**Interdit** : Impossible
- Repurgateur : Aucune magie
- Soldat : Magie interdite

## Impact lore (Arcanes uniquement)

**Dans domaine specialise** : Cout normal (CN × 10)
**Hors domaine** : IMPOSSIBLE (corruption si melange Vents)

**Exemple** : Sorcier Feu peut apprendre sorts Feu uniquement, jamais Bete/Cieux/etc.

## Composants additionnels

**Magie Naturelle** : +5 sous cuivre OU DR+1 Herboristerie par incantation
**Sorcellerie** : Sacrifices animaux (NI sous cuivre) sinon Imparfaites Mineures
**Chaos** : Composants couteux (NI pistoles argent), optionnels
**Arcanes Couleurs** : Composants optionnels (reduisent Imparfaites)

## Exemples calcul complet

**Sorcier Feu apprend "Boule Feu" (CN 5)** :
- Carriere Sorcier : 5 × 10 = 50 XP
- Change vers Erudit : 5 × 20 = 100 XP
- Change vers Soldat : Impossible (interdit)

**Initie Sigmar apprend benediction** :
- Carriere Initie : 50 XP
- Change vers Noble : 100 XP

**Initie Sigmar apprend miracle Sigmar** :
- Carriere Initie : 100 XP
- Change vers Noble : 200 XP

**Herboriste apprend sort naturel** :
- Carriere Herboriste : 30 XP + composants (5 sous cuivre/incantation)
- Change vers Charlatan : 60 XP + composants

**Sorcier Feu veut apprendre "Forme Bete" (domaine Bete)** :
- IMPOSSIBLE (melange domaines = corruption)

## Regles coeur

1. Arcanes : Cout = CN × 10 (carriere) ou CN × 20 (hors)
2. Divine : Fixe 50 (benedictions) ou 100 (miracles), double si hors carriere
3. Hors carriere toujours double
4. Hors domaine Arcanes impossible (pas cout, impossible)
5. Composants Naturelle obligatoires en plus XP
6. Chaos toujours hors carriere Empire (illegal)
