# Wizard - Détails personnage

## Vue d'ensemble

Étape détails : informations descriptives personnage (nom, physique, histoire).

## Champs

**Identité** : Nom.

**Physique** : Âge, Taille, Poids, Yeux, Cheveux.

**Histoire** : Ambition courte, Ambition longue, Marques distinctives.

**Motivations** : Liste motivations personnage (optionnel).

## Génération automatique

Génération aléatoire disponible pour : Âge, Taille, Poids, Yeux, Cheveux (selon espèce).

Distributions : Voir [../../database/species.md](../../database/species.md).

## Règles métier

Détails non bloquants (optionnels).

Génération aléatoire optionnelle (boutons individuels).

Modifications libres à tout moment.
