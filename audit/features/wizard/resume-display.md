# Récapitulatif Complet du Personnage

## Contexte

Écran résumé (dernière étape wizard) : vue complète personnage, organisée onglets.

## Organisation

### En-tête

10 caractéristiques principales : Nom, Valeur initiale, Avances, Total.

### Onglet "Perso"

Identité : Nom, race, signe, classe, carrière, niveau, âge, taille, détails.

Attributs dérivés : Destin/Chance, Résilience/Détermination, Mouvement, PB, Corruption.

### Onglet "Compétences/Talents"

Compétences base + groupées + avancées : Nom, carac associée, avances, total.

Talents (rang > 0) : Nom, rang/max, description.

### Onglet "Possession"

Possessions générales : Liste trappings.

Armures : Nom, localisation, enc, PA, qualités.

Armes : Nom, groupe, enc, portée/allonge, dégâts, qualités.

### Onglet "Sorts"

Affiché si personnage a sorts.

En-tête : Dieu (si Béni) ou Domaine magie (si Magie Arcanes/Chaos).

Liste sorts : Nom, type, CN, portée, cible, durée, effet.

## Affichage conditionnel

Onglet Sorts : Uniquement si sorts présents.

Dieu/Domaine : Selon talents magiques.

## Actions

Boutons : Valider (finaliser wizard), Sauvegarder (persister base), Annuler/Retour, Export PDF.
