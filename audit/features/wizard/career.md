# Wizard - Sélection carrière

## Vue d'ensemble

Étape carrière : choix carrière niveau 1, détermine classe sociale, compétences/talents/équipement départ, caractéristiques développables.

## Modes sélection

**Choisir** : Sélection manuelle parmi carrières compatibles espèce/région.

**Lancer** : Tirage aléatoire d100 selon table espèce.

## Génération aléatoire

Distribution selon espèce et région (Humains).

Voir [../../patterns/pattern-generation-aleatoire.md](../../patterns/pattern-generation-aleatoire.md).

## Application

Carrière niveau 1 choisie → application :
- Classe sociale (Académiques, Bourgeois, Citadins, Guerriers, Itinérants, Ruraux)
- Compétences niveau 1 (parsing, ajout origines)
- Talents niveau 1 (parsing, ajout origines, rangs multiples)
- Équipement classe + carrière
- Caractéristiques développables (celles améliorables pour cette carrière)

Parsing : Voir [../../patterns/pattern-parsing.md](../../patterns/pattern-parsing.md).

## Règles métier

Carrière obligatoire pour continuer wizard.

Niveau 1 uniquement en création. Progression ultérieure niveaux 2-4 post-création.

Carrières filtrées selon espèce et région (Humains).
