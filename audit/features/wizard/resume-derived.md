# Résumé - Attributs dérivés

## Vue d'ensemble

Calcul attributs dérivés finaux : Mouvement, Blessures, Destin, Résilience.

## Mouvement

Valeur base espèce : Humains 4, Nains 3, Elfes 5, Halflings 3, Gnomes 3, Ogres 6.

Modificateurs talents : Course à pied +1, Véloce +1.

Dérivés : Marche = Mvt × 2, Course = Mvt × 4.

## Blessures

Formules selon espèce :
- Humains/Halflings : 2×BE + BFM
- Nains : BF + 2×BE + BFM
- Elfes/Gnomes : 2×BE + BFM
- Ogres : (BF + 2×BE + BFM) × 2

Modificateurs talents : Endurci +BE/rang, Dur à cuire +1/rang, Très fort +1.

Bonus = Caractéristique ÷ 10 arrondi bas.

## Destin et Fortune

Destin : Valeur espèce + modificateurs signe + talents Chanceux/Né sous bonne étoile.

Fortune : Points consommables = Destin initial, régénération début session.

Affichage : Fortune courante / Destin max.

## Résilience et Détermination

Résilience : Valeur espèce + modificateurs signe + talent Imperturbable.

Détermination : Points consommables = Résilience initiale, régénération début session.

Affichage : Détermination courante / Résilience max.

## Encombrement

Total = Somme enc trappings.

Limite portage = Bonus Force × 10.

Seuils :
- Normal : enc ≤ BF × 10
- Surchargé : enc ≤ BF × 20 (Mvt ÷2, Ag -10)
- Immobilisé : enc > BF × 20

Voir [../../business-rules/calcul-encombrement.md](../../business-rules/calcul-encombrement.md).

## Corruption

Valeur initiale : 0 en création.

Accumulation post-création : Sorts Chaos, exposition, talents Chaos.
