# Wizard - Sélection Signe Astral

## Vue d'ensemble

Étape signe astral : choix signe naissance, bonus +3 caractéristiques principales.

## Modes sélection

**Choisir** : Sélection manuelle parmi 12 signes.

**Lancer** : Tirage aléatoire d100, +25 XP si accepté sans relance.

## Génération aléatoire

Tirage d100 : chaque signe probabilité différente.

Voir [../../patterns/pattern-generation-aleatoire.md](../../patterns/pattern-generation-aleatoire.md).

Bonus XP : +25 uniquement si acceptation premier tirage.

## Effets signes

Chaque signe : bonus +3 sur 1 ou 2 caractéristiques principales.

Exemples :
- Wymund le Résolu : +3 Force
- Mummit le Fou : +3 Intelligence
- Grunghus le Boeuf : +3 Endurance

Voir [star-effects.md](./star-effects.md) et [../../database/stars.md](../../database/stars.md).

## Règles métier

Signe optionnel (pas bloquant pour continuer).

Si omis : pas bonus caractéristiques.

Choix modifiable tant que wizard pas finalisé.
