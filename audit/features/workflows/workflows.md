# Workflows - Processus complets

## Vue d'ensemble

Workflows complets : création personnage, progression XP, changement carrière.

## Workflow création complète

11 étapes wizard :
1. Espèce (sélection/aléatoire)
2. Carrière (sélection/aléatoire)
3. Classe sociale (auto selon carrière)
4. Détails (nom, physique, histoire)
5. Signe astrologique (sélection/aléatoire)
6. Caractéristiques (tirages 2d10)
7. Compétences (affichage, spécialisations)
8. Talents (affichage, spécialisations)
9. Équipement (dotation carrière)
10. Expérience (dépense XP bonus)
11. Résumé (validation finale)

Validation : Chaque étape validable avant passage suivante, retour arrière possible, sauvegarde finale.

## Workflow progression XP

Process post-création :
1. Gain XP (récompenses MJ)
2. Ouverture interface advancement
3. Sélection améliorations (carac/compétences/talents)
4. Dépense XP (avances temporaires)
5. Validation (consolidation permanente)
6. Sauvegarde personnage

Calculs : Coûts selon paliers, multiplicateur ×2 hors carrière, validation budget.

Passage niveau : Validation conditions rang → Coût 100 XP → Accès nouveaux avantages niveau suivant.

## Workflow changement carrière

Process :
1. Vérification pré-requis (recommandations MJ)
2. Sélection nouvelle carrière
3. Confirmation changement
4. Coût XP (100-300 selon validation rang/classe)
5. Mise à jour niveau 1 nouvelle carrière
6. Recalcul coûts dans/hors carrière
7. Accès nouveaux avantages
8. Sauvegarde

Conservation : Caractéristiques, compétences, talents, sorts, XP dépensé.

Changements : Carrière actuelle, liste dans/hors carrière, coûts futurs, status niveau 1.
