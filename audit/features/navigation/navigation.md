# Navigation - Système navigation

## Vue d'ensemble

Système navigation application : arborescence, compendium, recherche, filtres.

## Arborescence

Organisation : Hiérarchique par catégories (Espèces, Carrières, Compétences, Talents, Sorts, Équipement).

Affichage : Tree view avec expansion/collapse, indication nombre éléments, navigation rapide.

## Compendium

Catalogue : Index complet toutes entités référentielles.

Organisation : Par type, alphabétique, avec compteurs.

Recherche : Filtre texte temps réel, recherche partielle, suggestions.

Affichage : Liste résultats, clic → détails élément popup/panneau.

## Filtres

Types : Par catégorie, par source (livre), par disponibilité.

Application : Temps réel, combinaison filtres, persistance sélection.

Affichage : Compteurs résultats filtrés.
