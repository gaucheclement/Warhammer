# Settings - Préférences utilisateur

## Vue d'ensemble

Gestion préférences utilisateur : affichage, validation, options interface.

## Préférences disponibles

Affichage : Thème (clair/sombre), langue, densité informations.

Validation : Règles MJ (validation rang carrière activée/désactivée), blocages XP création (strict/souple).

Interface : Animations, transitions, sons, tooltips.

Avancé : Mode debug, logs détaillés.

## Stockage

LocalStorage : Préférences sauvegardées navigateur, persistance sessions.

Format : JSON clé-valeur.

Défauts : Valeurs par défaut si aucune préférence.

## Application

Temps réel : Changements appliqués immédiatement, pas rechargement page.

Validation : Valeurs valides, types corrects.
