## Notes techniques

### Workflow complet
Ces exemples couvrent les 11 étapes du wizard de création :
1. ✓ Espèce
2. ✓ Carrière
3. ✓ Classe sociale (auto)
4. ✓ Détails (nom, physique, histoire)
5. ✓ Signe astrologique
6. ✓ Caractéristiques (tirages 2d10)
7. ✓ Compétences (distribution espèce + carrière)
8. ✓ Talents (choix 1 talent carrière)
9. ✓ Équipement
10. ✓ XP (bonus signes aléatoires)
11. ✓ Résumé (validation finale)

### Calculs appliqués

**Caractéristiques** :
- 15 obligatoires : 10 principales + M + PB + Destin + Résilience + Corruption
- Base espèce + Roll 2d10 + Bonus signe + Bonus talents
- Corruption toujours à 0 en création

**Points Blessures** :
- Formule par espèce : BF + 2×BE + BFM (Humain/Elfe/Nain)
- Halfling : 2×BE + BFM (pas de BF)

**Destin et Résilience** :
- Valeur de base par espèce (voir [../../database/characteristics.md](../../database/characteristics.md))
- Extra Points à distribuer librement entre Destin et Résilience
- Fortune = Destin, Détermination = Résilience (pools qui se rechargent)
- Talents peuvent modifier : Chanceux +1 Destin

**Compétences espèce** :
- 8 compétences acquises automatiquement
- Distribution : 3 à +5, 3 à +3, 2 à +0
- Valeur = Caractéristique liée + Avances

**Compétences carrière** :
- Budget : 40 points exactement
- Maximum : 10 points par compétence
- Cumul avec avances espèce si compétence commune

**Talents** :
- Espèce : Tous acquis automatiquement
- Carrière : **1 SEUL choisi** parmi ceux du rang
- Signe : Talent gratuit si applicable
- Rangs précisés si applicable (ex: Chanceux 1/3)

**Sorts** :
- Si talent addMagic (Magie mineure, Béni, etc.)
- Nombre = Bonus FM pour Magie mineure
- Gratuits en création (0 XP)

**XP bonus** :
- +25 XP si signe astrologique tiré aléatoirement et accepté
- +20 XP si espèce tirée aléatoirement et acceptée
