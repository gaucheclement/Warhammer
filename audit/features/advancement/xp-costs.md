# Advancement - Coûts XP

## Vue d'ensemble

Affichage coûts XP pour acheter améliorations personnage.

Voir [../../business-rules/calculs-xp-progression.md](../../business-rules/calculs-xp-progression.md) pour formules détaillées.

## Affichage coûts

### Caractéristiques

Coût calculé selon palier actuel.

Exemple : Force 45 → prochain +1 coûte 190 XP (palier 41-45).

Multiplicateur ×2 si hors carrière (affiché rouge).

### Compétences

Coût acquisition :
- Basic dans carrière : 10 XP
- Basic hors carrière : 20 XP
- Advanced dans carrière : 20 XP
- Advanced hors carrière : 40 XP

Coût avances : Calculé selon palier (10-380 XP).

Multiplicateur ×2 si hors carrière.

Spécialisations : Chaque = compétence distincte.

### Talents

Coût : 100 XP/rang (200 XP/rang hors carrière).

Spécialisations : Chaque = talent distinct.

### Sorts

Coût calculé selon formule magie.

Voir [../../business-rules/calculs-xp-progression.md](../../business-rules/calculs-xp-progression.md).

## Multiplicateur hors carrière

Indicateur : ×2 affiché rouge si hors carrière.

Changement dynamique : Recalculé lors changement carrière.
