# Task: Build Display Mode Wrapper Components

## Description
Create three wrapper components that adapt the EntityDescription viewer for different display contexts: modal (full-screen overlay), side panel (persistent reference), and inline (accordion-style expansion in lists). Each wrapper integrates EntityDescription with appropriate layout and behavior.

## Acceptance Criteria
- [ ] `DescriptionModal.svelte` component for full-screen overlay display
- [ ] `DescriptionPanel.svelte` component for slide-out side panel
- [ ] `DescriptionInline.svelte` component for inline accordion expansion
- [ ] Modal integrates with existing `Modal.svelte` base component
- [ ] Panel has resizable width (drag handle)
- [ ] Inline mode has expand/collapse animation
- [ ] All modes work with EntityDescription component
- [ ] Navigation controls display correctly in each mode
- [ ] Responsive behavior on mobile (320px+ width)
- [ ] Keyboard shortcuts: Esc (close modal/panel)

## Technical Details
- **New files**:
  - `src/components/DescriptionModal.svelte` - Full-screen modal wrapper
  - `src/components/DescriptionPanel.svelte` - Side panel wrapper
  - `src/components/DescriptionInline.svelte` - Inline accordion wrapper
- **Reuse**: `Modal.svelte` for modal base functionality
- **Opening API**: `openModal('description', {type, id})` for modal
- **Panel state**: Store open/closed and width in `ui.js` store
- **Animations**: Slide-in for panel, expand/collapse for inline

**Layout Specifications**:
- Modal: Full viewport overlay, centered content, dark backdrop
- Panel: Fixed right side, 400px default width (300-800px resizable)
- Inline: Embedded in list item, smooth height animation
